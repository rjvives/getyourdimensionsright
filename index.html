Measure Mapper MVP

Type a measurement and instantly get human-friendly comparisons (football fields, Olympic pools, garden hoses, city blocks, etc.). This is a tiny starter you can publish on GitHub Pages and expand with photos, videos, or animations.

Area

Examples used: Soccer pitch ~ 0.00714 km², Central Park ~ 3.41 km², Cologne Old Town ~ ~1.5 km² (approx).

Flow / Discharge

Examples used: Bathtub ~150 L, Garden hose ~ 15 L/min, Olympic pool ~ 2,500 m³.

Length / Distance

Examples used: Marathon 42.195 km, Cologne Cathedral height 157 m, City block ~100 m (varies).

This is a beginner-friendly starter. All math is done in the browser (no backend). You can extend the comparisons[] list in the script to add your own photos, videos, or animations later.
